Create Chimera Attribute Files.

You have a nice visualisation program. You have nice data. You want to visualise this data. You'd think it would be straightforward but often transforming your data to the format required by the visualisation program is a chore. Adun can atomatically create attribute files for the visualisation program Chimera, which as well as being powerful also produces very pretty pictures.
