Running a Simulation

Once you've created a simulation you'll want to start it going - read on.

To start a simulation:

  1. First create a simulation (see Creating a Simulation for more information)
  2. Select the created simulation in the process table
  3. Select 'Process->Start'. The process status will change from 'Waiting' to 'Running'
    • You can control the process using the other commands in the Process menu - try the Status command
  4. Once the simulation is finished an entry for it will appear in the database. However you can analyse the running simulation by selecting the entry in the process table and clicking Object->Analyse.
